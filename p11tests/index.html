SmartCard-HSM PKCS#11 Tests

Installation

To use the test suite, you must first install the Smart Card Shell.

Usage

You can start the test suite using

>load("p11-tests/loadtests.js");

Tests can be selected using the right mouse button on the test suite, test group or test case node.

Before running the tests you may want to change the location of the PKCS#11 module in the loadtest.js file.

Use Cases

The following tests are defined

  • Enumerate Objects
  • Signing
  • Decryption
